# Ish Malik — Physics & Engineering Portfolio

M.S. in Applied Physics and Engineering with a focus on optical systems, image sensing, and computational methods. This repo collects my projects in optical simulation, computational physics, and signal processing.

## Projects

### DVS Power Model *(in progress)*
Physics-based power model for Dynamic Vision Sensors (DVS), parameterized from the Lichtsteiner 2008 128×128 sensor. Models static vs dynamic power as a function of scene velocity, object size, background texture, and contrast threshold. Part of a broader CIS vs DVS sensor comparison study.

`Python` · `Sensor modeling` · `Event-based vision`

### [Fourier Optics Simulation](./fourier-optics-simulation/)
Simulates space-based vs ground-based telescope imaging using real Hubble Space Telescope data (M51 galaxy). Models diffraction-limited (Airy) and seeing-limited (Gaussian) PSFs, applies FFT-based convolution, and quantifies resolution loss via radially-averaged power spectra.

`MATLAB` · `FFT` · `PSF modeling` · `Fourier optics`

---

### [FFTLog Algorithm](./fftlog-algorithm/)
Python implementation of the FFTLog algorithm — a method for computing Hankel/Fourier transforms on logarithmically-spaced data. Implements the multiplier function using Bessel and Gamma functions, and validates reconstruction against known input.

`Python` · `NumPy` · `SciPy` · `Signal processing`

---

### [Helmholtz Resonator Analysis](./helmholtz-resonator/)
Real-time acoustic analysis tool for measuring Helmholtz resonance. Records live microphone input, computes spectrograms and power spectral density, and identifies peak resonant frequencies. Implemented in both Python and MATLAB.

`Python` · `MATLAB` · `SciPy` · `Acoustics` · `Signal processing`
